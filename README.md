## 📌 Projeto em Destaque

### 🎮 Pokédex Interativa

Uma aplicação inspirada no universo Pokémon desenvolvida para praticar conceitos de desenvolvimento web e organização de interfaces.

#### ✨ Funcionalidades

* 🔐 Tela de Login
* 📋 Listagem de Pokémon cadastrados
* 🔎 Tela de detalhes com informações completas do Pokémon selecionado
* ⚡ Catálogo de tipos Pokémon (Água, Fogo, Elétrico, Planta, Psíquico e outros)
* 📱 Interface intuitiva e responsiva

#### 🛠️ Tecnologia Utilizada

* Flutter/Dart

#### 🎯 Objetivos do Projeto

Este projeto foi desenvolvido para aprimorar conhecimentos em desenvolvimento web, manipulação de dados e criação de interfaces.
